vide — Examples

Browse the examples below. Open any HTML file directly in a browser after running pnpm build.

Getting Started

  • ui.html UI — Full UI plugin with default theme
  • ui-standalone.html UI — Manual component assembly (no convenience function)
  • ui-headless.html Headless — Custom theme without theme.css (cyberpunk style)
  • without-ui.html Without UI — Core player with native browser controls, no UI plugin

Streaming

  • hls.html UI — HLS adaptive streaming via hls.js
  • dash.html UI — DASH adaptive streaming via dash.js

Ads

  • ui-vast.html UI — VAST pre-roll ad with UI
  • hls-vast.html UI — HLS streaming + VAST pre-roll
  • dash-vast.html UI — DASH streaming + VAST pre-roll
  • vmap.html UI — VMAP ad schedule (preroll / midroll / postroll)
  • ad-pod.html UI — Ad pod sequencing and waterfall fallback
  • companion-ads.html UI — Companion ad slots (300x250, 728x90)
  • nonlinear-ads.html UI — NonLinear overlay ads
  • vpaid.html UI — VPAID 2.0 interactive ad creative
  • ima.html Without UI — Google IMA SDK pre-roll ad

Advanced

  • ssai.html UI — Server-side ad insertion (HLS + EXT-X-DATERANGE)
  • omid.html UI — Open Measurement (OM SDK) verification
  • simid.html UI — SIMID interactive ads in sandboxed iframe

CDN (no bundler)

  • cdn-all.html UI — All-in-one IIFE build (global Vide.*)
  • cdn-individual.html Without UI — Individual IIFE builds loaded separately
  • cdn-ads.html Without UI — CDN + HLS + VAST ads

Debug

  • vast-debug.html Debug — VAST XML inspection and manual ad playback
  • wrapper.html Debug — VAST wrapper chain resolution
