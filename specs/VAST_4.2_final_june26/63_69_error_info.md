## 6.9 Error Info

| Name | `[ERRORCODE]` |
|---|---|
| Type | `integer` |
| Introduced In | VAST 3.0 |
| Support Status | Required |
| Contexts | `Error` tracking pixels |
| Description | VAST Error Code. Replaced with one of the error codes listed in section 2.3.6.3 when the associated error occurs; reserved for error tracking URIs. |
| Example | `900` |
